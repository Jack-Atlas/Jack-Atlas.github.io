前言

买surface go 快一个月了，从上网收集资，观看各种评测，开箱的兴奋，上手的折腾，认真使用一段时间，再到回归我的y7000笔电，感触颇深，今天想用surface go 来写一篇读后感，但是使用体验让我敲下键盘选择写一篇surface go s简单评测以及使用小结

道听途说&收集资料

因为自己的专业是计算机相关，所以经常会关注数码圈，各种电子产品最新的评测都略有所知，加之自己自高中以来一直想构建自己的笔记系统，听说微软的onenote非常不错，有便捷高效的surface go 加持，使用应该会很不错，于是开始四处收集关于此类信息。

处理器

接口：3.5mm耳机接口×1，USB-C接口×1，此外便是微软专用的键盘盖接口×1，磁吸充电接口×1

续航大概为9小时
